# Salem_witch_trials_analysis
Machine learning analysis of historical data from the Salem witch trials (1692-1693)

## Overview
This project will explore whether or not demographic and historical can be used as information to predict the outcomes of the Salem witch trials. This research will strictly not make any historical claims about individual innocence or guilt, although it'll investigate if certain patterns existed according to records of the time.

## Research question
Are specific factors able to determine whether an accused person was executed?

## Dataset
Given the research questions, the dataset will be:
- age
- gender
- marital status
- occupation
- property ownership
- accuser count
- confession status
- trial outcome

## Objectives

- Create a structured dataset from historical records
- Analyze demographic and social patterns
- Apply statistical methods to historical data
- Develop a machine learning model to predict trial outcomes
- Visualize patterns found in the data
